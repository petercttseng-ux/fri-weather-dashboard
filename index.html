如何取得歷史氣象資料？
🤖
方法一：自動累積（推薦）
開啟本儀表板後每次點擊「更新」或自動更新，系統即自動將逐時資料存入本機。
設定自動更新間隔 10～30 分鐘，運行一段時間後即可累積歷史資料。
🌐
方法二：CWA 官網下載
從中央氣象署歷史資料查詢服務下載過去的觀測 CSV，再上傳至本系統。
雨量歷史資料查詢 開放資料平台
📂
方法三：上傳自備資料
若已有 CSV 或 JSON 格式的歷史氣象資料，可直接上傳。支援中英文欄位名稱。
上傳歷史資料檔案
支援欄位名稱：中英文皆可，時間欄為 ObsTime、DateTime 或 時間
CSV 欄位說明

雨量資料欄位：

ObsTime, StationId, StationName, County, Town,
Rain10Min, Rain1hr, Rain3hr, Rain6hr, Rain12hr, Rain24hr, Rain48hr

氣象資料欄位：

ObsTime, StationId, StationName, County, Town,
Temperature, RelativeHumidity, Pressure, WindSpeed, WindDirection, Precipitation
✓ 支援 YYYY-MM-DD HH:MM 及 YYYY/MM/DD HH:MM 時間格式
✓ 欄位名稱不區分大小寫，支援中文欄名
